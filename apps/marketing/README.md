# marketing — reserved path

The Caeorta marketing site is **not built in this monorepo**. It is hosted on
[Framer](https://www.framer.com/) (see `docs/03_Tech_Stack.md` → "Marketing site").

This directory exists only to reserve the `apps/marketing` path in the repo layout
(`docs/04_Repository_Structure.md`) and to point future contributors to Framer. There is no
Next.js/Expo project here and nothing to `pnpm install`.

If the marketing site ever moves in-repo (post-MVP), replace this README with the real app.
